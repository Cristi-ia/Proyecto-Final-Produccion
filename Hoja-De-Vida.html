Hoja De Vida

Cuatrimestre 1 • Producción Multimedial • 20/08/2025

Resultado Del Proyecto

El objetivo de este proyecto era colocar en práctica todo lo aprendido sobre HTML5 en clases, decorandolo con CSS3, hicimos una hoja de vida, la cual venia con varias secciones, fondo negro y por cada seccion tiene un marco de puntos, agregando una foto personal en la cual se uso una IA recomendada por el docente, tambien agregando links que llevan a mis páginas oficiales de redes sociales.

Herramientas Usadas

Todo lo aprendido en clases y lo practicado en Free Code Camp, usando el editor visual studio Code y publicandolo en Github
